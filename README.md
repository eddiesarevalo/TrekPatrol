# Trek Patrol

"Trek Patrol" is a fork of "Planet Patrol" which is a demonstration monitoring and notification system for the [Planetary Data System](https://pds.nasa.gov/). It keeps tabs of essential websites, APIs, and other online endpoints, files tickets when things aren't working, makes notifications (when configured to do so), and keeps things up and running, and neat and tidy.
